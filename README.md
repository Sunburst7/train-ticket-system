# train-ticket-system
基于Vue2与Springboot的火车售票系统

1. cd ./front

2. 启动前下载npm包
npm i

3. 启动前端:
npm run dev

4. 运行backend中的BackendApplication.java 启动后端

5. 别忘了导入数据库文件，但是数据库里的火车发车时间与到站时间需要修改
